# iTalento
iTalento es una plataforma de Freelance desarrollada por Kanto Software Solutions en NodeJS, ExpressJS y VueJS.

## Iniciar front
- cd front
- npm install

## Aplicar front (Siempre que se genere un cambio)
- npm run build 

## Iniciar Back
- cd back
- npm install
- npm run serve
